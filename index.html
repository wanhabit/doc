CARA MENGUNDUH

MENGUNDUH FILM DI WINDOWS

Pelajari cara mengunduh film dari Google Drive ke PC/Laptop Windows Anda.
Mengunduh di Browser Langsung
Cara paling dasar menggunakan browser web.
  • 1
    Buka Google Drive di browser dan login dengan Akun database film WAN-DB Anda (...@wanencode.com).
  • 2
    Klik bagian Shared Drive / Drive Bersama di menu sebelah kiri.
  • 3
    Masuk ke drive WAN-DB, dan cari folder film yang Anda inginkan.
  • 4
    Klik kanan pada file/folder dan pilih "Download / Unduh", atau tekan Shift + D.
  • 5
    Tunggu proses selesai, file akan masuk ke folder Download di komputer Anda. Enjoy!
Menggunakan Google Drive Desktop Recommended
Mengakses dan mengcopy file langsung melalui File Explorer Windows.
  • 1
    Unduh Google Drive untuk desktop di apps.wanhabit.com.
  • 2
    Instal aplikasi dan masuk menggunakan Akun database film WAN-DB Anda (...@wanencode.com).
  • 3
    Setelah sinkronisasi, buka File Explorer. Klik partisi Google Drive dengan ikon akun Anda.
  • 4
    Buka Shared Drive WAN-DB. Untuk mengunduh, cukup klik kanan file film lalu pilih "Copy".
  • 5
    "Paste" dikoasi penyimpanan lokal komputer Anda. Proses copy/download akan berjalan.
Menggunakan RaiDrive
Software penyimpanan awan pihak ketiga untuk Windows.
  • 1
    Buka aplikasi RaiDrive di komputer Windows Anda.
  • 2
    Tambahkan penyimpanan: Business > Google Shared Drive. Login dengan Akun database film WAN-DB (...@wanencode.com).
  • 3
    Buka File Explorer, masuk ke partisi RaiDrive yang baru saja dibuat.
  • 4
    Cari film, klik kanan "Copy" (Ctrl + C) lalu "Paste" (Ctrl + V) di penyimpanan lokal.

CLOUD STREAM (NONTON LANGSUNG)

CLOUD STREAM: WINDOWS

Menonton film tanpa perlu mengunduhnya. Langsung streaming dari cloud!
Menggunakan Aplikasi Kodi

Kodi disarankan jika Anda menginginkan pengalaman menonton layaknya Netflix atau bioskop pribadi di PC Anda, lengkap dengan poster, sinopsis, dan subtitle otomatis. Kami sudah menyediakan alat ekstrak agar instalasinya menjadi sangat mudah!

  • 1
    Persiapan Alat: Arahkan ke folder unduhan Kodi untuk Windows yang diberikan Admin. Ekstrak file .zip tersebut lalu buka foldernya.
  • 2
    Instalasi Otomatis: Di dalam folder, jalankan file bernama Setup.bat (klik 2 kali). Proses Command Prompt layar hitam putih akan memindahkan profil Kodi secara otomatis. Ikuti instruksi di layar dan tunggu hingga layarnya tertutup sendiri.
  • 3
    Mulai Aplikasi: Buka aplikasi Kodi dari Start Menu Windows Anda. Tampilan awal Anda kini seharusnya sudah dipenuhi oleh katalog film premium WAN-DB!
  • 4
    Autentikasi Perangkat: Coba klik salah satu film untuk mencoba menontonnya. Kodi akan menginfokan Anda untuk Login ke Google Drive (pilih Yes). Sebuah QR Code dan Kode Angka akan muncul di monitor Anda.
  • 5
    Tautkan Akun: Scan QR Code tersebut menggunakan HP (atau kunjungi manual ke link autentikasi). Ketikkan kode angka dari monitor PC ke layar HP/Browser Anda, lalu pilih login menggunakan Akun database film WAN-DB (...@wanencode.com) Anda dan berikan izin Allow.
  • 6
    Selesai! Kembali ke Kodi PC Anda. Film sekarang akan langsung diputar tanpa repot-repot mengunduhnya. Selamat menikmati pertunjukkan Cloud Streaming berkualitas terbaik di PC/Laptop Anda.
Menggunakan Google Drive Desktop Recommended
Paling praktis karena terintegrasi File Explorer dan Media Player (VLC / MPC-HC) langsung.
  • 1
    Pastikan Google Drive Desktop sudah terinstal dan Akun database film WAN-DB sudah tertaut.
  • 2
    Buka File Explorer, masuk ke partisi Shared Drive WAN-DB.
  • 3
    Cukup klik 2x (Double Click) file film tersebut. Secara otomatis film akan diputar di Media Player komputer Anda secara streaming!
    Catatan Media Player: Meskipun Windows sudah memiliki pemutar video bawaan, kami sangat menyarankan Anda untuk mengunduh dan memasang Media Player pihak ketiga seperti MPC-HC atau VLC. Aplikasi tersebut memberikan Anda fitur yang jauh lebih lengkap, termasuk kemampuan untuk mengganti Subtitle, memilih jalur Audio ganda, dan membaca format MKV premium kami dengan sempurna.
Fitur Ekstra Thumbnail Poster (Khusus Pengguna MPC-HC): Jika Anda mengkombinasikan Google Drive Desktop dengan pemutar video MPC-HC (K-Lite Codec Pack), Anda bisa melihat poster film layaknya galeri menempel langsung di ikon file pada Windows Explorer (ubah view ke Large/Extra Large Icons). Hal ini bisa terjadi karena format file kami adalah MKV murni premium yang sudah ter-embed/tersematkan metadata poster di dalamnya secara otomatis.

CLOUD STREAM (NONTON LANGSUNG)

CLOUD STREAM: ANDROID & TV

Menonton langsung dari Google Drive untuk pengguna Mobile dan Android TV.
Menggunakan Kodi Android & TV Sangat Disarankan

Pilihan terbaik untuk Android TV (Smart TV) atau HP Android. Kodi akan merubah TV Anda menjadi penampil khusus film.

Catatan: Penggunaan pada Android hanya akan menampilkan daftar folder dan file (tidak memuat tampilan poster/metadata gambar). Jika Anda ingin mendapatkan pengalaman mutlak menonton layaknya aplikasi Netflix lengkap dengan poster UI cantik, silakan jadilah langganan Quick Access kami melalui halaman Pembelian Akses.

  • 1
    Instal Aplikasi Asli: Buka Google Play Store di perangkat Android/TV Anda, cari aplikasi bernama Kodi, lalu tekan Install. Buka aplikasinya pertama kali dan tekan "Izinkan" (Allow) bila diminta akses penyimpanan.
  • 2
    Menambah Sumber (1/2): Pada halaman awal, klik ikon Gear (Pengaturan) di pojok kiri atas. Pilih menu File Manager, lalu klik dua kali pada tulisan Add Source.
  • 3
    Menambah Sumber (2/2): Akan muncul kotak dialog. Klik tulisan <None>, ketik secara persis alamat repositori resmi kami: https://wanencode.com/kodi/ (harus pakai strip/slash di belakang). Tekan OK. Nama sumber akan otomatis terisi menjadi "kodi", tekan OK lagi untuk menyimpan.
  • 4
    Persiapan Salin (1/2): Antarmuka layar terbagi menjadi panel direktori Kiri dan Kanan. Di panel sebelah Kanan, navigasikan dan klik masuk ke dalam Profile Directory. Pastikan Anda sudah masuk ke dalamnya.
    Wajib Diperhatikan: Panel kanan harus benar-benar masuk murni ke dalam Profile Directory. Jika tidak, opsi untuk meng-Copy folder pada langkah selanjutnya tidak akan pernah muncul di menu Kodi.
  • 5
    Persiapan Salin (2/2): Beralih ke panel sebelah Kiri, buka folder sumber kodi yang baru ditambahkan. Gunakan remot TV, tahan lama / klik kanan pada salah satu folder sampai keluar menu opsi, pilih Select All. Tahan lama lagi, lalu kali ini pilih Copy dan konfirmasi Yes.
    Catatan Copy: Aksi copy ini otomatis langsung menempelkan (paste) data ke panel kanan, tidak ada tombol paste. Jika progres mandek dan gagal mencapai 100%, mohon lakukan trik Copy satu-persatu per menu/folder tanpa menggunakan Select All.
  • 6
    Mulai Ulang (Restart) Kodi: Anda berhasil! Tutup sepenuhnya aplikasi Kodi / Force Close. Lalu buka aplikasinya kembali.
    PENTING: Tampilan utama awal akan terlihat sama seperti sebelumnya! Anda BUKAN mengecek di halaman awal, melainkan pergi ke menu Favourites (Favorit). Di sanalah daftar poster kategori yang tadi Anda copy berada.
  • 7
    Langkah Terakhir - Autentikasi: Klik salah satu kategori di menu Favorit. Akan muncul instruksi menginstal *Addon Google Drive*, tekan Yes sebanyak mungkin. Tunggu sesaat hingga muncul Popup QR Code & Link Tautan di TV Anda.
    ⚠️ Syarat Mutlak Internet: HP (untuk scan QR/Link) dan TV Anda WAJIB menggunakan internet dengan alamat IP yang sama persis (Misal: Gunakan koneksi dari Hotspot HP ke TV, atau koneksi WiFi 2.4Ghz yang sama). Jika ada tulisan error "IP Change", segera matikan Kodi, Clear Cache & Clear Data aplikasi Kodi, lalu ulangi dengan pastikan IP Internet sudah sama rata.
    Lakukan tautan akun di HP tersebut, dan login menggunakan Akun database film WAN-DB (...@wanencode.com) Anda. Selesai! Klik kategorinya lagi, dan seluruh folder judul film akan termuat. Selamat menikmati!

APP DRIVES

Penjelasan terkait Google Drive Desktop dan RaiDrive untuk Windows.
Rangkuman Software Drive
Untuk mengoptimalkan pengalaman menonton Cloud Stream di PC, wajib menggunakan Virtual Drive agar film bisa dimainkan di Media Player terbaik.

Google Drive Desktop menempelkan Drive langsung ke sistem Windows Anda. Sinkronisasi sempurna dengan Shortcut. Ini adalah yang paling disarankan.

RaiDrive adalah perangkat lunak pihak ketiga yang mengemulasikan Drive Anda melalui protokol koneksi jaringan. Berfungsi sangat baik sebagai alternatif ringan namun tidak mendukung pembacaan file bertipe 'Shortcut' Google Drive.

CARA TEMUKAN FILM TERBARU

Panduan wajib agar Update Film / Episode Series terbaru selalu muncul di urutan teratas.
Wajib: Urutkan Berdasarkan 'Date Modified'
Secara default, Google Drive dan File Explorer di Windows/Apple mengurutkan file berdasarkan Nama (Abjad A-Z). Karena kami mengupdate film setiap hari, Anda wajib mengubah urutannya agar film yang baru kami tambahkan tampil paling atas.
  • 1
    Jika menonton via Browser:
    Buka folder film di Google Drive. Di pojok kanan atas daftar file, klik tombol "Name" / "Nama" lalu ubah menjadi "Last modified" / "Terakhir diubah".
  • 2
    Jika menonton via Windows (Drive Desktop / RaiDrive):
    Buka folder di File Explorer. Klik Kanan di area yang kosong → pilih Sort by → pilih Date modified. Pastikan urutannya Descending (Terbaru ke Terlama).
  • 3
    Jika menonton via Aplikasi (Infuse / Kodi / nPlayer):
    Masuk ke pengaturan / opsi urutan (Sort) di dalam aplikasi tersebut, dan ubah pengurutannya menjadi Date / Time / Date Added.
Tahapan ini sangat krusial, terutama bagi Anda yang mengikuti serial TV *On-Going* (Berskambung). File episode baru yang kami tambahkan akan langsung bertengger di paling atas berkat fitur *Date Modified* ini.
